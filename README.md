
## Prerequisites

[https://github.com/foundry-rs/foundry](https://github.com/foundry-rs/foundry)

[https://web3py.readthedocs.io/](https://web3py.readthedocs.io/)

## Usage

### Install

```shell
forge install
```

### Build

```shell
forge build
```

### Test

```shell
forge test
```

### Run

```shell
python3 example.py
```
